# TDC001

## Actions

컴퓨터에 연결된 kinesis 기기를 찾고, 특정 범위를 특정 번 반복하여 움직일 수 있도록 합니다.  

## Caution

visual studio 환경에서 개발이 된 프로젝트이기 떄문에 해당 IDE기준으로 셋팅이 완료되어 있습니다.  
Thorlabs에서 본 프로젝와 관련된 "모든" 파일(일반 dll 포함)을 ./bin 폴더에 넣어주어야 작동을 합니다.
